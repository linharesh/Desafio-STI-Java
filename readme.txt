Desafio implementado em java, utilizando o Netbeans 8.1
Para executar o projeto, é necessário ter o JRE instalado na máquina.
O JRE pode ser baixado gratuitamente através do link http://www.oracle.com/technetwork/java/javase/downloads/server-jre8-downloads-2133154.html
Com o JRE instalado, basta abrir a pasta "Executável JAR" e abrir o arquivo "Desafio.jar"
